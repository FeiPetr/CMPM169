Experiment 2: Vector Art, Animation & Interactivity

Imitate

I have very little experience with JavaScript beyond Phaser 3, so this is all very new to me. I looked through some existing code to try to understand them.

Starting off, I was especially intrigued by various text based interactive projects. I copied the code from P_3_2_1_01 by generative-design and began to modify the code to get a better understanding of how text based interactive projects worked. I began to modify the original code and experiment with it to create my own interactive text project.

On top of modifying the look of the text, I decided to try and add additional features, such as the ability to hit return and write more on a new line. However, I spent a lot of time on this with very little progress, so I decided to move onto something new.

Integrate

I began looking at color generative projects. In particular, I was intrigued by P_1_1_1_01 by generative-design, where a color gradient would expand according to mouse movement.

I decided to experiment with an interactive color background combined with my text experiment. After integrating into my project, I now have a gradient background that updates with every letter typed rather than automatically moving with the mouse, with a stroke in between colors.

I experimented with the color gradient with the goal of being able to manipulate it creatively. I ended up modifying the step function in a way that made an interesting combination of refresh rates where parts of previous writing would be kept depending on where the mouse was.

Innovate

I began to utilize random color generation for both the background and the letters being typed. The colors update at each keystroke. I created a sort of glitchy background in a new project using the knowledge of background color creation used in the previous project.

After this, I incorporated typing back into it by stretching all the letters to create a more fitting alien-esque font. On top of this, I added random color generation to the letters so that the letters changed color with every keystroke. I decided to conclude my project here, having run out of time and creative ways to further it.
